GraleJ


Version 0.3.2

A platform independent viewer for attribute-value matrices that can be used as a graphical display for TRALE.

Authors

GraleJ was written by Martin Lazarov, Armin Buch, and Niels Ott. This project is funded by the Collaborative Research Center 441 (Sonderforschungsbereich 441) of Tübingen University.

Credits

GraleJ uses the Tomato parser written by Martin Lazarov.

For SVG output, GraleJ uses the Batik SVG Toolkit.

The crystal icon theme is based on the Crystal SVG theme from KDE 3.x, which is brought to the open source world by Everaldo.

The traditional icon theme was adapted from the KDE 2.x default theme by Holger Wunsch.
